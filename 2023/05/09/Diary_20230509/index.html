回顾40天（4.1-5.9）

发表于 2023-05-09 分类于 日志 阅读次数：
本文字数： 758 阅读时长 ≈ 1 分钟

前言

40天的生活，完全打乱了之前的节奏。

源头

说起源头，那就是因为机缘巧合，做起了代付YouTube频道会员的业务。
一开始的预期是，能有十几位来找我都不错了，因为之前完全没做过，对于预期管理得很低。
但是后来随着不断宣传，发现人越来越多，一开始准备的资金完全不够，应对客户的经验完全不够，实操过程中遇到的问题也很多。
就这样，即便跌跌撞撞，我还是靠着起初市场的巨大空缺，成交了3位数的客户。
想起当初一天几十位客户，应付起来是真的非常累。从起床到睡觉，饭都顾不上吃。

影响

由于这个业务的开展，导致了我的生活发生了巨大变化。
先说说好的变化：

  1. 更熟悉网络。对于YouTube付款机制遇到的问题，基本都能找到解决办法。
  2. 认识了各种各样的人。对于人这方面，有了更深入的了解。
  3. 获取了收益。让我的生活变得更加从容。

再来说说不好的变化：

  1. 生活习惯完全打乱。每天都不能早睡早起，午睡时间也非常乱。
  2. 注意力变差。由于客户很多，为了以防错失客户，经常会检查有没有新消息。
  3. 专业方面落下。本来就学得勉勉强强，又因为做了这个业务，导致专业的东西没碰多少。
  4. 生活节奏混乱。原本每天的规律，已经不复存在，想要纠正回来，也是十分困难。

总的来说，
这样的交换，是否好呢？
目前我还无法看清，姑且判断为等价交换吧。不算值得，也不算太亏。

选择

实际上，要以此为起点，做成主业，并非是不可能。
但我不想选择这种，竞争性门槛低的工作。
这种事，只要了解了其中的流程，谁都能做到。
被人抢走市场份额，那是迟早的事。
我不想长期做这种看不到未来的事。
况且，这种业务是基于变化很大的网络。即便专业，也随时可能颠覆。

今后

既然选择了仅仅把这个业务当作是赚取零花钱的副业，那么其重要性就不高了。
掂清轻重，所以要学会选择。
今天写这篇，也是为了下定决心。
早睡！早起！
从明天开始，就尽全力恢复以前的生活节奏吧。

我很可爱，请给我钱
