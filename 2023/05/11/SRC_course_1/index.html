Burp使用

  1. 过滤心跳包
    （1）使用代理插件（如SwitchOmega），将不想抓包的网址放入。
    （2）或者使用Burp，过滤一些不需要显示的数据包。

  2. 爆破模块
    想要对比返回包的时候，可以将Response的内容发送到Comparer。然后选择Words或byte比对。

  3. app抓包
    iPhone教程，在WiFi栏配置代理，然后配置与Burp一样的IP和端口。
    配置好后，在浏览器上打开IP和端口，将Burp的配置文件下载到手机上。
    之后在“通用”->”VPN与设备管理”->安装该配置文件
    最后测试能否抓包。可以则证明该手机型号成功，不可以则证明该手机型号不可抓包。

tips:
抓包的时候乱码，说明做了app加固，或是Burp中设置的字体有问题。

  1. 模拟器抓包
    官网下载地址： http://mumu.163.com
    BurpSuite设置监听局域网内IP（这个IP是电脑当前IP，端口任意选择）
    MuMu模拟器那边设置（设置-WLAN-长按WiFi名-修改网络-填好，保存）
    代理为之前填写的ip+端口
    然后用模拟器自带浏览器访问BurpSuite的页面（ip+端口），下载证书
    再使用自带文件管理，找到下载的证书，将文件后缀改为.cer
    修改后缀方式：长按所要修改的文件即可。
    安装成功后，在受信任的凭据（设置-安全-信任的凭据）内看到刚刚安装的证书
    打开app，打开网站，都成功抓包。证明成功。

  2. 微信小程序抓包
    建议直接用电脑版微信抓包即可，核心思路是设置全局代理。
    win注意事项：
    微信抓包找到这个文件
    C:\Users\HoAd\AppData\Roaming\Tecent\WeChat\XPlugun\Plugins\WMPFRuntime
    删掉这个目录的文件，然后重启微信。
    Mac:知识星球里

dvwa的安装

使用 docker serach 关键字，输入想找的镜像，然后再pull下来，这里选择
citizenstig/dvwa:
docker pull citizenstig/dvwa

创建容易（此处将容易重命名了为dvwa）：
docker run --name dvwa -d -p 9001:80 citizentig/dvwa

访问该地址后，先初始化数据库。点击Create/Reset Database即可

tips:
抓包的话，可以使用电脑本机IP打开DVWA，这样即可抓包。
