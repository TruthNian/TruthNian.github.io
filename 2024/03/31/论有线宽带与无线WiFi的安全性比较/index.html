论有线宽带与无线WiFi的安全性比较

发表于 2024-03-31 分类于 硬件安全 阅读次数：
本文字数： 4.5k 阅读时长 ≈ 4 分钟

前言

因为不太了解硬件安全方面，最近在纠结这个问题。所以写了一篇文章来记录。

宽带与WiFi概述

宽带即采用有线的方式传输数据。
PPPoE协议的工作流程包含发现和会话两个阶段，发现阶段是无状态的，目的是获得PPPoE终结端（在局端的ADSL设备上）的以太网MAC地址，并建立一个唯一的PPPoE SESSION-ID。发现阶段结束后，就进入标准的PPP会话阶段。

WiFi即采用无线的方式连接。
WiFi普遍的定义是一种允许电子设备连接到一个无线局域网（WLAN）的技术，有些WiFi设置成加密状态，也有些WiFi是开放的、不需要密码即可接入的。最常见的WiFi信号来自于无线路由器，无线路由器如果本身连接到了互联网，那么它也可以被称为热点。
AP（Access Point）即相当于无线路由器的作用，有线网络通过无线路由器等设备，自行建立一个无线局域网环境，使得具备无线接入功能的设备以无线方式接入路由器内网，通过路由器实现上网功能。
AP以每100ms一次的频率由Beacons（信号台）将无线网络的SSID（Service Set Identifier）发射出去，确保覆盖范围内所有的无线客户端能够收到这个SSID广播封包，并依据无线客户端的需求决定是否要和此AP进行网络连接。

GPT4对于宽带与WiFi的安全性看法

在网络安全和保密的角度看，有线宽带和WiFi上网各有其优缺点。以下是从安全性和保密性的角度对两种网络连接方式的详细比较：

有线宽带上网

优点：

  1. 物理安全性：有线连接需要物理接入点，这意味着攻击者需要实际接触到网络线缆或设备才能访问网络。这增加了未经授权访问的难度。
  2. 更难被拦截：与无线信号相比，有线信号更难被远程拦截。攻击者需要对物理线路进行接触，才能尝试截取数据传输。
  3. 无需加密：虽然加密仍然是提高数据传输安全性的好方法，但有线网络由于其物理特性，相对于无线网络来说，即使未加密也更难被外部访问。
  4. VLAN隔离：在较大的有线网络中，使用虚拟局域网（VLAN）技术可以提高网络的安全性，通过逻辑上分隔网络来限制广播域和隔离敏感数据流。
  5. 网络隔离和控制：在有线网络中，更容易通过物理端口和虚拟局域网（VLAN）来实现网络隔离，从而限制攻击者的横向移动。如果入侵被限制在特定的网络段内，那么潜在的损失可能会减少。
  6. 监控和检测：有线网络可能更容易进行流量监控和异常检测，因为网络管理员可以更精确地控制和审计通过物理端口的流量。

缺点：

  1. 灵活性有限：有线网络的使用受限于物理连接。在需要频繁移动设备或在没有预先布线的区域提供网络接入时，可能会遇到困难。
  2. 部署成本：对于大型建筑或多层办公楼，布设有线网络可能需要更多的时间和成本，特别是在需要钻孔或铺设大量网络线缆的情况下。
  3. 端口安全：在有线网络中，物理接入点（如以太网端口）可能成为安全漏洞。未经授权的设备可以通过物理方式接入网络。因此，端口安全（如802.1X认证）对于防止未经授权的接入至关重要。
  4. 物理安全漏洞：虽然物理接入更难，但不可忽视物理安全漏洞，如未锁好的网络机房或易受攻击的网络基础设施。
  5. 物理访问：如果攻击者通过钓鱼攻击获得了网络内部主机的控制权，并且能够进一步获得物理访问权限，那么他们可能会尝试对网络设备进行物理篡改，如安装监听设备。

WiFi上网

优点：

  1. 便利性和灵活性：WiFi提供了无线网络连接，极大地提高了设备的移动性和网络接入的便利性。用户可以在网络覆盖范围内自由移动设备。
  2. 易于部署：在没有物理网络基础设施的环境中，部署WiFi网络相对简单快捷。
  3. 网络隔离：使用访客网络（Guest Network）功能可以为访客提供网络接入，同时将其流量与内部网络隔离，保护内部网络不受潜在的恶意行为影响。
  4. 灵活性：WiFi网络提供了更大的灵活性，使得在发生入侵时，快速变更网络配置（如更改SSID和密码）相对容易，这可能有助于迅速隔离受感染的设备，减少损失。

缺点：

  1. 信号泄露：无线信号可以穿透墙壁和窗户，可能被建筑物外的未经授权的设备接收到，增加了被监视和攻击的风险。
  2. 加密和安全设置：为了保护数据传输的安全性，WiFi网络需要配置加密和安全措施（如WPA2或WPA3加密）。不正确或过时的配置可能导致安全漏洞。
  3. 更易受到某些攻击：WiFi网络相对更容易受到中间人攻击（MITM）、网络钓鱼和邪恶孪生（Evil Twin）等攻击。
  4. 需要额外的安全措施：为了提高WiFi网络的安全性，可能需要采取额外的安全措施，如设置强密码、定期更换密码、隐藏SSID、限制MAC地址访问等。
  5. WPS漏洞：WiFi保护设置（WPS）旨在简化网络的安全设置，但在某些实现中存在安全漏洞，可能允许攻击者绕过WiFi密码。
  6. 信号覆盖控制：无线信号的覆盖范围应控制在需要服务的区域内，避免信号泄露到不需要的区域（如街道或邻近建筑），减少潜在的攻击面。
  7. 固件更新：无线路由器和接入点的固件应保持最新，以修补已知的安全漏洞。
  8. 横向移动：在WiFi网络中，攻击者可能更容易在未经充分隔离的网络内进行横向移动，寻找其他脆弱目标。
  9. 信号泄露：WiFi信号的泄露可能使得攻击者在没有物理接入的情况下，通过无线方式继续探测和攻击网络内的其他设备。

当黑客通过钓鱼或其他远程攻击手段成功入侵到主机时，他们获得的访问权限和能够造成的损失通常与网络连接类型（有线宽带或WiFi）关系不大。

个人看法

如果说流量控制和隔离的话，有线网络也能通过相应的网络设备和策略来实现类似的功能。
如果说是隐藏SSID和白名单mac，想要破解也不是无法做到。

  1. 突破MAC地址过滤
    MAC地址是网络设备的唯一网络地址标示，本意为媒体访问控制，可以理解为网络设备的物理地址，对于每台网络设备，MAC地址是固定且唯一的，采用十六进制数表示，共六个字节。在前篇文章介绍的破解过程中，无线网卡、终端、路由器等设备的MAC地址，都是我们用来进行破解的重要数据。
    MAC地址过滤防护机制，即通过预先在无线路由器内设定允许或者禁止访问本地网络的设备MAC地址列表，以白名单/黑名单的方式来控制网络设备的接入权限。这项安全机制在有线路由器和无线路由器中均存在，并不以连接方式有区别。路由器在接收到网络接入请求后，根据接入终端的MAC地址对照本地过滤设置规则中的MAC地址，判定是否允许网络终端进行接入行为。
    如果本地网络的网络终端比较固定，最好采用“允许”即白名单的方式，固定化终端MAC地址，也就是本地网络中只允许特定的几个终端进行接入。这种方式比“禁止”的黑名单方式安全性更高，但使用灵活度低。无论是白名单还是黑名单，路由器对接入终端的权限判断都基于MAC地址这单一选项，这就给攻击者创造了一种突破思路。
    主要的突破思路，就是通过修改本地的网络设备MAC地址，绕过目标路由器的过滤名单达到突破的目的。前文我们讲到，MAC地址虽然理论上唯一，但可以通过软件修改等方式，将本地设备的MAC地址临时修改为所需要的MAC地址。突破的步骤可以分为上图所示的四个步骤：
    •探测合法无线客户端的MAC地址，可以使用airodump-ng工具实现
    •修改攻击网卡的MAC地址，这个步骤可以在Windows或Linux中进行
    •重新装载网卡设备，并检查是否已经更改成功
    •连接无线接入点，完成突破防护的目的

  2. 突破隐藏无线网络SSID
    在无线路由器的设置中，如果目标使用了隐藏SSID选项，那么正常情况下，打开手机、平板的无线网络，是搜索不到目标的网络名称的。这种情况下只有知道了网络SSID名称和密码并且手动添加输入才能够连接。但是要特别强调，关闭SSID广播功能仅仅是关闭了无线路由器广播信标帧中的SSID部分，并不代表无线路由器的无线信号就不发射和接收了。虽然SSID被隐藏，但是无线路由器的MAC地址以及其他信息还是能够探测到的。
    第一种使用airodump-ng和aireplay-ng进行DeAuth攻击获取SSID。在将无线网卡加载激活至监听模式后，使用airodump-ng进行信号探测。
    第二种通过被动探测的方式进行SSID获取，比较流行的一种工具就是kismet，这款工具也在Kali中集成了。被动探测相对于主动探测，隐蔽性更好，而且探测通信更加可靠。某些无线路由器对于主动探测，可以配置为不回复将SSID设置为“任意”的探测请求帧，若为被动探测，只有在有合法客户端与无线接入点进行连接时才能探测到SSID，因为只有产生连接认证时，SSID才能以明文方式出现在无线电信号中。

在一个密室里面，人员进出都经过了专门检查，那么即使是近源，也应该接触不到宽带。而WiFi的话因为信号暴露则会更容易。

措施

如果使用宽带，则需要在密室，访问人员进出都需要经过验证。需要配置VLAN隔离。
如果使用WiFi，则需要修改SSID名称并隐藏SSID，密码设置为包含数字大小写字母特殊字符16位以上（最好每月更改一次密码），白名单MAC地址，使用WPA3协议，启用路由器防火墙。

启用路由器防火墙方法

在路由器上启用防火墙通常涉及登录到路由器的管理界面并修改安全设置。不同品牌和型号的路由器可能会有不同的界面和选项，但大多数路由器都提供了某种形式的防火墙功能。以下是一般步骤：

1. 访问路由器管理界面

  1. 找到路由器的IP地址：通常是192.168.1.1、192.168.0.1或类似地址。你可以在路由器的底部标签或手册中找到这个信息。如果不确定，也可以通过电脑的网络设置查看默认网关地址。

  2. 登录：在浏览器的地址栏输入路由器的IP地址，并按回车键。这将打开路由器的登录页面。输入管理员用户名和密码进行登录。如果你未曾更改过这些信息，它们可能是出厂设置，通常也可以在路由器的手册或底部标签上找到。

2. 启用防火墙

  1. 查找防火墙设置：登录后，查找安全性或防火墙设置的选项。这可能在“高级设置”、“安全”或“防火墙”等部分。

  2. 启用防火墙功能：在防火墙设置中，应该有一个选项来启用或禁用防火墙。确保这个选项被激活。在一些路由器上，防火墙可能默认就是开启的。

  3. 配置防火墙规则（可选）：一些高级路由器允许你配置具体的防火墙规则，比如阻止或允许特定的入站和出站流量、端口或IP地址。如果你对网络安全有特定需求，可以根据需要调整这些设置。

  4. 保存设置：更改设置后，记得保存更改。路由器可能需要重新启动才能应用新的设置。

3. 测试防火墙

  • 测试连接：更改设置后，测试网络连接确保一切正常工作。你可以尝试访问几个网站，确保防火墙设置没有意外阻止正常的网络流量。

注意事项

  • 默认密码：出于安全考虑，如果你还在使用路由器的默认管理员密码，强烈建议更改为一个强密码。
  • 固件更新：确保路由器固件是最新的。旧的固件可能包含未修复的安全漏洞，定期检查并应用固件更新是保持网络安全的重要措施。
  • 备份配置：在进行重大更改之前，备份当前的路由器配置是一个好习惯。这样，如果新设置导致问题，你可以恢复到之前的状态。

通过以上步骤，你可以在路由器上启用并配置防火墙，以增强网络的安全性。如果在配置过程中遇到问题，查阅路由器的用户手册或联系制造商的技术支持可能会有所帮助。
